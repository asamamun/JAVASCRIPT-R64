super() is a function that calls the constructor of the parent class. In JavaScript, when you use class inheritance (extends), the child class must call super() inside its constructor before accessing this.

Why is super() Required?
The super() function executes the parent class constructor.
The this keyword is not available in a child class until super() has been called.
