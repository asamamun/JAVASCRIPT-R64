OOP: Object Oriented Programming

Javascript is a OOP language from version ES6, javascript supports class from version ES6

Class in javascript

class is a blueprint for creating objects

class can have properties and methods

class can have constructor

To create a class we use the keyword class

Object in javascript

object is an instance of a class

object can have properties and methods

object can have constructor

To create an object we use the keyword new

Constructor in javascript

constructor is a function that is called automatically when an object is created

constructor is used to initialize the object

constructor is used to set the properties of the object

constructor can call other methods of the class and it can also call parent class constructor

Methods in javascript

method is a function that is called on an object

method is used to set the properties of the object

method is used to get the properties of the object

method is used to call other methods of the class and it can also call parent class method

methods are functions. functions inside a class are called methods. like lau and kodu
