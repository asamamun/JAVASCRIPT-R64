A JavaScript object is like a container that holds related data and functionality. Think of it as a real-world object, say a car: it has characteristics (like color or model) and things it can do (like drive or honk). In JavaScript, those characteristics are properties, and the things it can do are methods. For example: array object has properties like length and methods like push, pop, join,shift,unshift, slice, splice, concat etc.

Creating a JavaScript Object

Objects are created using the keyword object followed by a name for the object in curly braces {}. 

            
                let car = {
                    brand: "Toyota",          // property
                    model: "Camry",           // property
                    year: 2020,               // property
                    isRunning: false          // property
                  };
