JavaScript: The TraineeForm Class

The TraineeForm class is defined in the <script> section and handles form submission and data processing. Let’s examine its components:

1. Constructor

constructor(formId) { this.form = document.getElementById(formId); this.form.addEventListener("submit", (event) => this.handleSubmit(event)); }
  • Purpose: Initializes the class by linking it to the form element with the given formId.
  • What it does:
    • this.form stores a reference to the DOM element (e.g., the form with ID traineeForm).
    • Adds a submit event listener to the form, which calls handleSubmit when the form is submitted.
  • Arrow function: The (event) => this.handleSubmit(event) syntax ensures this inside handleSubmit refers to the TraineeForm instance.

2. getFormData Method

javascript
getFormData() { return { name: this.form.name.value, contact: this.form.contact.value, remarks: this.form.remarks.value, sex: this.form.sex.value, courses: Array.from(this.form.courses).filter(c => c.checked).map(c => c.value), location: this.form.location.value }; }
  • Purpose: Collects data from the form fields.
  • How it works:
    • this.form.name.value: Gets the value of the input with name="name".
    • this.form.contact.value: Gets the value of the input with name="contact".
    • this.form.remarks.value: Gets the value of the textarea with name="remarks".
    • this.form.sex.value: Gets the value of the checked radio button with name="sex".
    • this.form.courses: Refers to all checkboxes with name="courses".
      • Array.from(this.form.courses): Converts the NodeList to an array.
      • .filter(c => c.checked): Keeps only the checked checkboxes.
      • .map(c => c.value): Extracts their values (e.g., "NT", "IAD").
    • this.form.location.value: Gets the selected option from the dropdown.
  • Returns: An object with the form data.

3. formatData Method

javascript
formatData(data) { return ` <h2>Trainee Details</h2> <p><strong>Name:</strong> ${data.name}</p> <p><strong>Contact:</strong> ${data.contact}</p> <p><strong>Sex:</strong> ${data.sex}</p> <p><strong>Location:</strong> ${data.location}</p> <p><strong>Courses:</strong> ${data.courses.join(", ")}</p> <p><strong>Remarks:</strong> ${data.remarks}</p> `; }
  • Purpose: Formats the form data into an HTML string for display.
  • How it works:
    • Takes the data object from getFormData.
    • Uses template literals to create a readable HTML structure.
    • data.courses.join(", "): Converts the courses array into a comma-separated string (e.g., "NT, IAD").
  • Returns: A string of HTML.

4. handleSubmit Method

javascript
handleSubmit(event) { event.preventDefault(); //form jeno submit na hoy const data = this.getFormData(); const newWindow = window.open("", "_blank", "width=400,height=400"); newWindow.document.write(this.formatData(data)); newWindow.document.close(); }
  • Purpose: Handles form submission.
  • How it works:
    • event.preventDefault(): Prevents the default form submission (which would reload the page).
    • this.getFormData(): Retrieves the form data.
    • window.open("", "_blank", "width=400,height=400"): Opens a new browser window (400x400 pixels).
    • newWindow.document.write(this.formatData(data)): Writes the formatted HTML into the new window.
    • newWindow.document.close(): Finalizes the document writing process.
