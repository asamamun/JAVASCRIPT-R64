Operators can be 3 types: unary, binary and ternary operators

unary operators

  • ++: increment ex. x = x + 1 can be written as x++
  • --: decrement ex. x = x - 1 can be written as x--
  • typeof: returns the type of a variable. ex. typeof(x) or typeof "hello"
  • instanceof: returns true if an object is an instance of a class, ex. x instanceof String
  • void: returns undefined, ex. void(x)
  • delete: deletes a property, ex. delete x.
  • new: creates a new object. ex. new String("hello")
  • in: returns true if a property exists in an object, ex. "name" in x
  • ... : spread operator, spread the values of an array
  • ... : rest operator, get the rest of the values

binary operators

  • == : equal, ex. x == y
  • != : not equal. ex. x != y
  • === : strict equal, ex. x === y. it checks if the value and the type are same and returns boolean value
  • !== : strict not equal. ex. x !== y. it checks if the value and the type are same and returns boolean value
  • > : greater than
  • < : less than
  • >= : greater than or equal
  • <= : less than or equal
  • += : add. ex. x += y is x= x + y
  • -= : subtract. ex. x -= y is x= x - y
  • *= : multiply. ex. x *= y is x= x * y
  • /= : divide. ex. x /= y is x= x / y
  • %= : modulus. ex. x %= y is x= x % y
  • **: power operator x**y is xy, ex: 2**3 is 8

ternary operators

  • ?: : ternary operator. syntax: condition ? true : false, ex: age >= 18 ? "you are eligible to vote" : "you are not eligible to vote"
