ECMAScript 2015, also known as ES6, introduced JavaScript Classes.

Class: JavaScript Classes are templates for JavaScript Objects.

How to create class in javascript

        
            example 1: class className {
                
              }

            example 2: class className {

            }

Objects: Instances/example of a class. objects are created using new keyword

Constructor: special method(function) that is called when an object is created
