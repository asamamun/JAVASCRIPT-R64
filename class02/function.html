function: a block of code that performs a specific task when it is called

general syntax to create a function

        
            function functionName(parameters) {
                //code to be executed
            }

general syntax to call a function

        
            functionName(parameters)

function can be 2 types: built-in and user-defined

  • Built in popular functions: alert(), confirm(), prompt(), Number(), parseInt(), parseFloat()
  • User defined functions
